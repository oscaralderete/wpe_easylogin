# Módulo Personalizado de Drupal: EasyLogin

Este es un ejercicio para desarrolladores junior de Drupal, ¡evita subirlo a tu servidor!

Existen muchos tutoriales sobre cómo programar Drupal, pero la mayoría se enfocan únicamente en las interacciones con PHP. En este módulo sencillo puedes aprender a usar JavaScript puro para una interacción básica en el frontend.

Drupal utiliza mucho jQuery, pero si eres un desarrollador principiante probablemente conozcas más JS que jQuery. Con este módulo verás cómo trabajar con Drupal evitando jQuery. Es super fácil.

Además, podrás ver cómo crear una página de administración para configurar tus credenciales y así tener un acceso rápido cuando necesites iniciar sesión.

## Instrucciones

Clona el repo directamente en el directorio de Drupal los módulos custom:

```bash
cd web/modules/custom
git clone https://github.com/oscaralderete/wpe_easylogin.git
```

Liampia la caché y habiñlita el módulo:

```bash
drush cr
drush en wpe_easylogin
```

si 'drush' no es directamente invocable, usa './vendor/bin/drush':

```bash
./vendor/bin/drush cr
./vendor/bin/drush en wpe_easylogin
```

# Drupal Custom Module: EasyLogin

This is an exercise for junior Drupal developers, avoid to upload it to your server!

There are many tuts about how to master Drupal but all of them are focused only in PHP interactions, in this simple module you can learn how to use pure JavaScript for simple frontend interaction.

Drupal uses jQuery a lot but if you're a newbie developer probably knows more JS than jQuery so in this module you can see how to deal with Drupal avoiding jQuery. It's really easy.

Besides you can see how to create an admin page to set your creentials for an easy access when you need to log in.

## Instructions

Clone the repository directly into your Drupal custom modules directory:

```bash
cd web/modules/custom
git clone https://github.com/oscaralderete/wpe_easylogin.git
```

Clean the cache and enable the module:

```bash
drush cr
drush en wpe_easylogin
```
